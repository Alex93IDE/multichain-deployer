# Despliega contratos inteligentes en cualquier blockchain compatible con EVM y solidity

## Install the dependencies
```bash
yarn
# or
npm install
```

## Create .env

```bash
MNEMONICS=
WALLET_ADDRESS=
ADDRESS_INDEX=0
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
